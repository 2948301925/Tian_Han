# Tian_Han

- 该APP是一款基于 uni-app 框架开发的汉中景点展示小程序核心页面。
- 集成了景点搜索、轮播图展示、景点卡片列表展示等功能。
- 用户可通过搜索关键词快速筛选心仪的汉中景点。
- 点击景点卡片可跳转至对应详情页。
- 整体界面美观、交互流畅，适配多端运行（微信小程序、App、H5 等）。

## 🌟 核心特性

1. 导航栏与搜索功能
- 集成uni-nav-bar导航栏，包含状态栏适配，搜索框采用圆角设计，聚焦时边框变色，交互体验佳；
- 实时搜索：输入关键词时自动过滤景点列表，支持按景点标题和描述模糊匹配；
- 搜索为空时自动恢复展示全部景点，无匹配结果时显示 “未找到相关结果” 提示。
2. 轮播图展示
- 自动轮播：配置 5 秒自动切换，切换动画时长 300ms，激活态指示器为绿色；
- 图片适配：采用aspectFill模式，保证图片比例不失真，宽度 100% 适配屏幕。
3. 景点列表展示
- 卡片式布局：使用uni-card组件展示景点信息，包含标题、描述、缩略图、操作按钮；
- 交互效果：卡片点击时有缩放反馈，提升操作感知；
- 区域标题：突出 “汉台区” 核心景点分类，视觉层级清晰。
4. 页面跳转
- 统一跳转方法：点击景点卡片调用uni.navigateTo跳转到对应景点详情页，路径配置在景点数据中。

## 📸 代码结构

1. 模板层（template）
- 导航栏模块：包含搜索框、搜索图标、右侧搜索按钮；
- 轮播图模块：循环渲染轮播图片列表；
- 景点列表模块：循环渲染过滤后的景点卡片，无结果时展示空状态；
- 核心指令：v-for循环渲染、v-model绑定搜索关键词、v-if控制空状态显示。
2. 逻辑层（script）
- 数据定义：搜索关键词、轮播图列表、全量景点数据、过滤后景点数据；
- 生命周期：mounted初始化时展示全部景点；
3. 样式层（style）
- 采用 SCSS 预处理器，定义全局样式变量（如导航栏高度、颜色等），便于统一维护；
- 响应式设计：使用 rpx 单位适配不同屏幕尺寸；
- 交互样式：搜索框聚焦边框变色、卡片点击缩放、阴影效果提升视觉层次。

## 核心方法：

- handleSearch：实时过滤景点，支持标题 / 描述模糊匹配；
- goPage：统一处理页面跳转，兼容 uni-app 多端路由规则。

## 🚀 快速开始
### 环境要求

- 框架：uni-app（Vue2/Vue3）；
- 组件：uni-ui（uni-nav-bar、uni-card、uni-icons）；
- 预处理器：SCSS（需在 uni-app 中开启 SCSS 编译）。

### 使用说明

- 将代码集成到 uni-app 项目中；
- 配置静态图片路径（/static/目录下放置对应景点图片）；
- 配置详情页路由（确保url字段对应的页面路径存在）；
- 运行到对应平台（微信小程序需配置 appid，H5 需配置跨域等）。
